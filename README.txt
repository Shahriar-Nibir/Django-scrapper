Installing dependencies:
1. Open command promt in this directory write:

pip install -r requirements.txt


2. for running:

python manage.py runserver


http://127.0.0.1:8000           will take you to home.
http://127.0.0.1:8000/scrap     will scrap the data.

All the data are already scrapped. On running http://127.0.0.1:8000/scrap url it will add new 
posts. You can also delete all the posts and browse http://127.0.0.1:8000/scrap url to scrap 
again.
